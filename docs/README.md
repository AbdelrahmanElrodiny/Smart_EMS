# docs

Put your project media here, then reference it from the main `README.md` Gallery section.

Suggested files:

- `hardware.jpg` — photo of the assembled hardware (breadboard / enclosure)
- `dashboard.png` — screenshot of the web dashboard at `http://192.168.4.1`
- `display.jpg` — photo of the DWIN display showing the sensor/clock pages
- `demo.gif` — short screen recording or clip of the system running (optional, high impact)

Keep images reasonably sized (under ~1–2 MB each) so the repo stays light and the README loads fast.
